INTRODUCTION
-----
UnitXは，生活向けの計算機言語（プログラミング言語）です．
数値に単位(時間，Byte数など)に着目した言語です．

INSTALL
-----
	make
	sudo make install

HOW TO USE
-----
Run Demo program like below.

	make demo0

TEST
-----
Run XUnit, if you wanna try it.
	make unittest
